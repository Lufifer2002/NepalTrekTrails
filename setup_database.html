Database Setup for Nepal Trek Trails

Setup Instructions

This page will help you set up the database for the Nepal Trek Trails application. Please follow these steps:

  1. Make sure XAMPP is running and Apache is started
  2. Click the "Run Database Setup" button to create the necessary tables
  3. Click the "Insert Sample Data" button to add sample packages
  4. After setup, you can visit the dashboard to see the packages loaded from the database
